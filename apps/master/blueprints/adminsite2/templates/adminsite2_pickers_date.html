Date Pickers

The pickadate.js is a mobile-friendly, responsive, and lightweight jQuery date & time input picker which allows for high levels of flexibility than any other plugin.

The basic setup requires targetting an input element and invoking the picker.

Change the month and weekday labels as you find suitable.

Change the text or hide a button completely by passing a false-y value.

Change the titleattributes to several elements within the picker.

The picker can be extended to add support for internationalization.

Display a human-friendly format and use an alternate to submit to the server.

A majority of the time, the value that needs to be sent to the server is just the hidden value – and not the visible one. To make this happen, use the hiddenName option.

By default, inputs are readOnly. Change this by using editable: true option.

Display select menus to pick the month and year.

You can also specify the number of years to show using an even integer.

Set the minimum and maximum selectable dates on the picker.

Disable a specific or arbitrary set of dates selectable on the picker.

Fire off events as the user interacts with the picker.

When a date is selected or the “clear” button is pressed, the picker closes. To change this behavior, use the different options as specified during the initializing of pickadate component.
