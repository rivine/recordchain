Time Pickers

The plugin pickatime.js is a highly flexible JavaScript plugin for various kinds of time pickers.

This is an example of default time picker.

The picker can be extended to add support for internationalization.

Change the text or hide the button completely by passing a false-y value.

Display a human-friendly label and input format and use an alternate one to submit.

Choose the minutes interval between each time in the list.

Disable a specific or arbitrary set of times selectable on the picker.

Set the picker to respond to user actions.

Fire off events as the user interacts with the picker. Open your console to see results.
